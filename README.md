# FotoShop IA 🛍️

Gerador de fotos de produtos para Shopee e Mercado Livre usando Google Gemini.

## Como usar
1. Acesse o site pelo link do GitHub Pages
2. Insira sua chave da API do Google AI Studio
3. Descreva seu produto, escolha o estilo e gere a imagem!

## Chave gratuita
Obtenha em: https://aistudio.google.com/apikey
